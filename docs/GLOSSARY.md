# 凡人修仙模拟器 — 术语词典

> 按分类整理了游戏内所有专有名词与核心术语。约 2,000+ 字。

## 境界类

| 术语 | 说明 |
|------|------|
| **炼气** | 修仙第一步，吸收天地灵气淬炼己身。1-10 层，寿元 +30 年。 |
| **筑基** | 构筑修仙根基，灵气在丹田中凝聚为液态真元。11-20 层，寿元 +100 年。 |
| **金丹** | 真元凝结为金丹，体内自成小天地。21-30 层，寿元 +300 年。 |
| **元婴** | 金丹破而元婴生，可短暂离体遨游。31-40 层，寿元 +800 年。 |
| **化神** | 元神大成，可感知天地法则。41-50 层，寿元 +2000 年。 |
| **炼虚** | 返虚合道，虚实之间随意转换。51-60 层，寿元 +5000 年。 |
| **合体** | 肉身与元神完美合一。61-70 层，寿元 +12000 年。 |
| **大乘** | 功德圆满，只待天劫降临。71-80 层，寿元 +30000 年。 |
| **渡劫** | 九重天劫，渡则飞升，败则魂飞魄散。81-90 层。 |

## 系统类

| 术语 | 说明 |
|------|------|
| **气运** | 影响奇遇触发率和突破成功率的基础属性。上限 100 点。 |
| **业力** | 击杀怪物和玩家累积的负面值。业力过高时触发天劫概率增加。 |
| **天劫** | 大境界突破时有概率触发的惩罚性事件。渡劫成功可获得额外属性加成。 |
| **心魔** | 突破失败产生，降低修炼效率。可通过清心丹或特殊奇遇消除。 |
| **灵力** | 基础战斗资源，世界频道发言消耗灵力。 |
| **真元** | 施展功法技能消耗的核心资源，受丹田等级影响。 |
| **修为** | 升级所需的经验值。通过挂机修炼、战斗、任务获得。 |
| **战力** | 综合评价角色实力的数值。受装备、功法、境界、灵根等多因素影响。 |
| **行动力** | 采集消耗资源。上限 200，每 10 分钟恢复 1 点。 |
| **好感度** | NPC 和玩家之间的亲密度。等级：陌生、友好、信任、亲密、挚交。 |
| **贡献度** | 完成宗门任务获得，用于学习宗门技能和兑换物品。 |
| **亲密度** | 玩家好友之间的亲密值。达到一定等级解锁特殊功能。 |
| **灵石** | 游戏基础货币，玩家间交易使用。 |
| **绑定灵石** | 不可交易货币，仅用于系统商店和强化。 |
| **仙玉** | 充值货币，购买稀有物品和抽卡。 |
