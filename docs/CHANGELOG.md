## v1.0.0 (2026-06-05) - 正式版

### 新增功能

#### 境界体系
- 完整九境十层90级等级体系: 炼气、筑基、金丹、元婴、化神、炼虚、合体、大乘、渡劫
- 每层修为需求动态曲线，前期(炼气~金丹)快速升级，后期(合体~渡劫)长线耐心
- 突破系统: 每层概率突破机制，支持丹药、气运、护法多种加成
- 突破失败惩罚机制，含修为扣除和冷却时间
- 突破动画特效和天劫雷云表现

#### 挂机修炼系统
- 自动修炼: 在线自动积累修为值
- 离线收益: 最多累计12小时离线修炼收益(月卡用户延长至24小时)
- 灵气潮汐: 每6小时刷新世界灵气状态，影响全体修炼效率
- 洞府系统: 三种档次洞府租赁，提供修炼场景加成

#### 战斗系统
- 39种妖兽: 覆盖九大境界的8个难度梯度的怪物图鉴
- 43种法术技能: 五行(金木水火土) + 雷风变异系 + 无属性技能体系
- 五行克制系统: 金克木->木克水->水克火->火克土->土克金，克制1.3倍/被克0.7倍
- 境界压制机制: 跨境界战斗时高境界方获得压制加成
- 暴击/闪避/命中三要素属性平衡
- 世界Boss和区域Boss刷新机制

#### 炼丹系统
- 31种丹方: 涵盖恢复、修炼、属性、特殊四大类
- 5档品质浮动: 下品/中品/上品/极品，品质影响效果数值
- 9种破境丹: 每个大境界对应专属破境丹，分品质提供5%~20%突破加成
- 材料采集: 14个区域分布不同等级采集物
- 炼丹成功率系统: 受丹炉品质、炼丹等级、材料品质影响

#### 世界探索
- 14个区域: 从青牛镇到上古祭坛，每个区域特色鲜明
- 16种奇遇事件: 随机触发，含选择分支和不同奖励结果
- 8个NPC: 分布在各个区域，提供任务、传授、交易等功能
- 传送系统: 已探索区域间快速传送(消耗灵石)

#### 宗门系统
- 创建宗门: 5级宗门体系，人数上限随等级提升
- 宗门技能: 5种宗门技能可升级，惠及全体成员
- 宗门任务: 每日刷新宗门任务，获得宗门贡献和灵石
- 宗门战: 每周跨服宗门对战，获胜宗门获得灵脉占领权
- 宗门资金: 成员捐献、宗门战奖励，用于升级和技能研发

#### 经济系统
- 灵石(基础货币) + 仙玉(付费货币) 双货币体系
- 坊市交易: 玩家间自由交易，系统收取5%交易税
- 拍卖行: 竞价拍卖稀有物品，3%成交手续费
- 6档充值档次: 6元~648元，赠送比例递增
- 商城: 修为丹、破境丹、灵石袋、皮肤箱等商品

#### 排行榜
- 修为排行榜: 全服修为值排名
- 战力排行榜: 综合战斗力排名
- PVP排行榜: 竞技场积分排名
- 财富排行榜: 总资产排名

#### 任务系统
- 主线任务: 10章50个任务，贯穿整个游戏历程
- 支线任务: 5条独立支线，丰富世界观
- 每日任务: 5个日常任务 + 活跃度宝箱系统

#### 气运系统
- 气运值: 范围0~1000+，影响突破概率、掉落率、奇遇品质
- 气运增减: 行善积德增加，杀戮作恶减少
- 业力系统: 业力值过高会导致突破惩罚

#### PVP竞技场
- 10段位体系: 凡人到渡劫真仙
- 赛季制: 每赛季60天，赛季末重置并发放奖励
- 属性平衡: PVP模式归一化属性，避免碾压

#### 外观系统
- 4套仙侠主题皮肤: 水墨风、金碧辉煌、暗黑修真、青莲剑仙
- 角色时装幻化系统
- 武器外观特效

### 技术架构

- **后端:** 9个Go微服务 (网关、认证、玩家、战斗、炼丹、世界、宗门、排行榜、聊天)
- **前端:** Vue 3 + TypeScript + Pinia 状态管理 + Vue Router
- **数据持久化:** MySQL(核心数据) + Redis(缓存/会话) + MongoDB(日志/聊天)
- **部署:** Docker Compose 一键部署 + Nginx 反向代理
- **通信:** WebSocket(实时战斗/聊天) + HTTP REST API
- **CI/CD:** GitHub Actions 自动化测试与构建
- **监控:** Prometheus + Grafana 指标监控

### 优化

- 服务端战斗计算优化至单次 <5ms
- 前端虚拟列表渲染千级好友/排行榜无卡顿
- WebSocket 连接断线自动重连机制
- 数据库读写分离，主从延迟 <100ms
- 静态资源 CDN 加速加载

### 修复

- 修复突破概率在某些边界条件下显示不一致的问题
- 修复离线收益在跨天时计算异常
- 修复拍卖行竞拍保证金退回逻辑
- 修复宗门贡献在退出宗门后未正确清零的问题
- 修复五行克制在特定组合下计算顺序错误

### 安全

- 全链路 HTTPS 加密
- 服务端数值校验防止变速齿轮/内存修改
- 敏感操作日志审计
- SQL注入防护参数化查询
- 交易日志全量记录用于反欺诈

## v0.9.0 (2026-05-20) - 内测版

### 新增功能

- **基础修炼系统:** 在线挂机修炼，离线收益(最多8小时)
- **突破系统:** 九境十层基础突破，含突破动画
- **PVE战斗引擎:** 实时回合制战斗，支持五行克制
- **背包系统:** 物品分类、堆叠、使用、丢弃
- **装备系统:** 6部位装备栏，白绿蓝紫金5品质，强化+9
- **WebSocket聊天:** 世界频道 + 私聊 + 系统频道
- **角色创建:** 随机灵根(7种品质) + 姓名生成
- **新手引导:** 青牛镇新手任务链，引导至炼气十层

### 技术架构

- Go 单体服务 (战斗 + 玩家 + 聊天)
- Vue 3 + TypeScript 前端
- MySQL + Redis
- WebSocket 双向通信

### 修复

- 修复战斗计算中防御减伤异常(减伤超过100%导致治疗反噬)
- 修复背包物品排序在不同客户端不一致
- 修复聊天消息重复推送问题
- 修复突破界面概率显示与实际计算不一致

## v0.5.0 (2026-04-01) - Alpha版

### 新增功能

- **核心架构搭建:** Go 微服务框架 + 网关层 + 服务注册发现
- **API网关:** 路由分发 + 限流 + JWT认证 + 请求日志
- **认证服务:** 手机号/邮箱注册登录 + 验证码 + 密码加密
- **玩家服务:** 角色创建 + 基础属性查询 + 升级
- **数据库表设计:** 玩家表、物品表、境界表等核心表结构
- **Docker Compose:** MySQL + Redis + Nginx + 3个Go服务
- **CI流水线:** GitHub Actions 代码检查 + 单元测试 + 构建

### 技术选型

- Go 1.22 + Gin 框架
- GORM + MySQL 8.0
- Redis 7.0 (Session/缓存)
- Nginx 反向代理
- Docker + Docker Compose

### 已知问题

- 无前端界面，仅提供 REST API 调试
- 无战斗系统，玩家创建后仅可查看属性
- 无持久化玩家数据备份方案
- 限流策略较为粗糙，未做精细化配置

## 版本号规范

本项目遵循语义化版本号规范: `主版本.次版本.修订号`

- **主版本:** 重大版本更新，可能包含不向后兼容的改动
- **次版本:** 功能更新，向后兼容
- **修订号:** Bug 修复和微小优化
