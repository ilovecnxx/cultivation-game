## 1. 境界成长曲线

### 1.1 整体架构

本游戏采用 **九境十层** 的等级体系，总计 90 级。每个大境界分为 10 个小层（第 1 层至第 10 层），每层需要积累足够的修为值方可突破至下一层。

### 1.4 成长节奏设计

**核心目标:** 普通玩家从炼气一层修炼至渡劫十层（满级）的时间控制在 **18~24个月**。

**各阶段耗时估算:**
| 境界 | 预估耗时 | 日均在线需求 | 备注 |
|------|----------|-------------|------|
| 炼气 (~10级) | 1~2 天 | 4小时 | 新手期, 快速升级引导 |
| 筑基 (11~20级) | 3~5 天 | 4小时 | 解锁功法系统 |
| 金丹 (21~30级) | 7~10 天 | 3小时 | 解锁炼丹系统 |
| 元婴 (31~40级) | 14~20 天 | 3小时 | 解锁宗门系统 |
| 化神 (41~50级) | 1~2 个月 | 2小时 | 开始需要团队 |
| 炼虚 (51~60级) | 2~3 个月 | 2小时 | 中期瓶颈 |
| 合体 (61~70级) | 3~4 个月 | 2小时 | 大量丹药辅助 |
| 大乘 (71~80级) | 4~5 个月 | 1.5小时 | 高难度突破 |
| 渡劫 (81~90级) | 5~6 个月 | 1.5小时 | 终极冲刺 |

**节奏设计要点:**
1. **前期快速反馈:** 炼气期1-2天即可通过，确保新玩家在前48小时内体验到核心玩法循环。
2. **中期稳定增长:** 金丹至元婴期保持每周1-2级的增速，维持玩家成就感。
3. **后期长线投放:** 合体期后每个小层需1-2周，配合月度版本更新投放新内容。
4. **离线收益补偿:** 离线最多累计12小时修炼收益，离线效率为在线效率的60%。
5. **加速机制:** 丹药可缩短30%-50%的修炼时间，付费仙玉可直接兑换修为（每日上限5次）。

## 2. 伤害公式

### 2.1 最终伤害计算公式

```
最终伤害 = 攻击力 × 技能倍率 × 五行修正 × (1 + 境界压制) × 暴击修正 - 目标防御 × 0.5
```

**参数说明:**
| 参数 | 说明 |
|------|------|
| 攻击力 | 角色面板攻击力（基础+装备+功法加成） |
| 技能倍率 | 技能自身倍率，范围 1.0 ~ 12.0 |
| 五行修正 | 根据五行克制关系计算，范围 0.5 ~ 2.0 |
| 境界压制 | 境界差带来的压制效果，每差1境界+10% |
| 暴击修正 | 暴击时 = 1.5~3.0，非暴击 = 1.0 |
| 目标防御 × 0.5 | 防御削减系数，防御只减半生效 |

### 2.2 五行克制表

**五行相克关系:**
| 攻击方 \ 防御方 | 金 | 木 | 水 | 火 | 土 |
|----------------|----|----|----|----|----|
| 金 | 1.0 | 1.3 | 0.7 | 1.0 | 1.0 |
| 木 | 1.0 | 1.0 | 1.3 | 0.7 | 1.0 |
| 水 | 0.7 | 1.0 | 1.0 | 1.3 | 1.0 |
| 火 | 1.0 | 0.7 | 1.0 | 1.0 | 1.3 |
| 土 | 1.3 | 1.0 | 1.0 | 0.7 | 1.0 |

**克制关系原理:** 金克木 -> 木克水 -> 水克火 -> 火克土 -> 土克金，克制方获得 1.3 倍伤害修正，被克方降低至 0.7 倍修正，同属性/无关属性为 1.0 倍。

### 2.3 境界压制系数表

境界压制仅在攻击方境界高于防御方时生效，同境界或低于时压制系数为 0。

| 境界差 | 压制系数 | 说明 |
|--------|----------|------|
| 0 | 0% | 同境界无压制 |
| 1 | 10% | 高1个小境界 |
| 2 | 20% | 高2个小境界 |
| 3 | 35% | 高3个小境界 |
| 4 | 50% | 高4个小境界 |
| 5 | 70% | 高5个小境界 |
| 6 | 90% | 高6个小境界 |
| 7+ | 120% | 超过7层以上压制封顶 |

**大境界压制倍率:**
| 大境界差 | 额外压制系数 | 叠加方式 |
|----------|-------------|----------|
| 同大境界 | 0% | 仅计算小境界差 |
| 差1大境 | 50% | 小境界压制 + 50% |
| 差2大境 | 120% | 小境界压制 + 120% |
| 差3大境+ | 250% | 小境界压制 + 250% |

**示例:** 金丹一层攻击筑基十层，小境界差 = 金丹一层(21级) - 筑基十层(20级) = 1级，大境界差 = 1。最终压制系数 = 10%(小境界) + 50%(大境界) = 60%。

### 2.4 暴击机制

```
暴击率 = 面板会心值 / (面板会心值 + 等级 × 50 + 500)
暴击伤害 = 基础暴击倍率(1.5) + 会心加成
```

**会心属性成长:**
| 境界 | 基础会心值 | 暴击率(估算) | 基础暴伤 |
|------|-----------|-------------|----------|
| 炼气 | 50 | 4.8% | 1.5 |
| 筑基 | 100 | 5.9% | 1.6 |
| 金丹 | 200 | 6.5% | 1.7 |
| 元婴 | 350 | 7.0% | 1.8 |
| 化神 | 500 | 7.1% | 1.9 |
| 炼虚 | 750 | 7.5% | 2.0 |
| 合体 | 1,000 | 7.7% | 2.2 |
| 大乘 | 1,500 | 8.3% | 2.5 |
| 渡劫 | 2,000 | 8.9% | 3.0 |

### 2.5 DOT与治疗公式

**持续伤害(DOT):**
```
每跳伤害 = 技能攻击力 × 0.3 × 持续时间(秒) / 总跳数
```

**治疗公式:**
```
治疗量 = 基础治疗 × 功法倍率 × (1 + 灵力/1000)
```

## 3. 突破概率公式

### 3.1 核心公式

```
最终成功率 = 基础成功率 + 丹药加成 + 气运加成 + 护法加成 - 业力惩罚 - 仓促惩罚
```

### 3.2 基础成功率

| 突破类型 | 基础成功率 | 说明 |
|----------|-----------|------|
| 小层突破(1~9层) | 85% | 同一大境界内的层数提升 |
| 大境突破(炼气->筑基) | 50% | 首次大境界突破 |
| 大境突破(筑基->金丹) | 45% | - |
| 大境突破(金丹->元婴) | 40% | - |
| 大境突破(元婴->化神) | 35% | - |
| 大境突破(化神->炼虚) | 30% | - |
| 大境突破(炼虚->合体) | 25% | - |
| 大境突破(合体->大乘) | 20% | - |
| 大境突破(大乘->渡劫) | 15% | 最高难度突破 |
| 渡劫十层->飞升 | 5% | 终极突破 |

### 3.3 丹药加成

**破境丹效果:**
| 品质 | 小层突破加成 | 大境突破加成 | 说明 |
|------|-------------|-------------|------|
| 下品 | +3% | +5% | 绿色品质 |
| 中品 | +5% | +10% | 蓝色品质 |
| 上品 | +10% | +15% | 紫色品质 |
| 极品 | +15% | +20% | 金色品质 |

**丹药叠加规则:** 同一种丹药效果不可叠加，不同种类的丹药效果可叠加（如破境丹+聚气丹可同时生效）。

### 3.4 气运加成

```
气运加成 = min(玩家气运值 / 100, 10%)
```

| 气运值 | 突破加成 | 档位 |
|--------|---------|------|
| 0~99 | 0% | 厄运 |
| 100~199 | +1% | 微吉 |
| 200~299 | +2% | 小吉 |
| 300~399 | +3% | 中吉 |
| 400~499 | +4% | 大吉 |
| 500~599 | +5% | 上吉 |
| 600~699 | +6% | 大吉 |
| 700~799 | +7% | 大吉 |
| 800~899 | +8% | 大吉 |
| 900~999 | +9% | 大吉 |
| 1000~ | +10%(上限) | 天命 |

### 3.5 护法加成

| 护法类型 | 加成值 | 条件 |
|----------|--------|------|
| 同境界道友 | +2% | 好友/同门关系 |
| 高1大境护法 | +5% | NPC或玩家 |
| 高2大境护法 | +10% | 需消耗人情值 |
| 高3大境+护法 | +15% | 需消耗宗门贡献 |
| 宗门护法阵 | +8% | 消耗宗门资源开启 |

### 3.6 业力惩罚

业力值通过杀戮其他玩家、抢夺资源、背叛宗门等行为获得。

| 业力值区间 | 突破惩罚 | 说明 |
|-----------|---------|------|
| 0~50 | 0% | 正常状态 |
| 51~100 | -3% | 轻度业力缠身 |
| 101~200 | -8% | 中度业力 |
| 201~500 | -15% | 重度业力 |
| 500+ | -25% | 业火焚身, 惩罚封顶 |

**业力清除方式:** 捐献灵石（100灵石/1业力）、完成善事任务、使用清心丹（丹药: 清除50业力）。

### 3.7 仓促惩罚

| 突破准备 | 惩罚 | 说明 |
|----------|------|------|
| 修为满后立即突破 | -10% | 无任何准备 |
| 修为满后稳定1天 | -5% | 稍作沉淀 |
| 修为满后稳定3天 | 0% | 充分准备 |
| 修为满后稳定7天 | +3% | 厚积薄发 |
| 修为满后稳定30天+ | +5% | 沉淀圆满 |

### 3.8 突破失败后果

| 失败类型 | 后果 |
|----------|------|
| 小层突破失败 | 修为不减, 1小时内不可再次突破 |
| 大境突破失败 | 修为扣除20%, 24小时内不可再次突破 |
| 大境突破失败(第3次+) | 修为扣除35%, 48小时内不可再次突破 |
| 渡劫飞升失败 | 修为扣除50%, 境界降至大乘一层, 7天冷却 |

## 4. 经济系统

### 4.1 货币体系

游戏采用 **双货币体系**:
| 货币 | 名称 | 获取方式 | 用途 |

| 灵石 | 基础货币 | 打怪, 任务, 交易 | 日常消耗 |
| 仙玉 | 付费货币 | 充值, 活动 | 高级消费 |

### 4.2 灵石产出

**怪物掉落:**
```
灵石掉落 = 怪物等级 × 5 + 随机(0, 怪物等级 × 3)
```

| 怪物等级区间 | 灵石掉落区间 | 掉落概率 |
|-------------|-------------|---------|
| 1~10 (炼气) | 5~40 | 80% |
| 11~20 (筑基) | 55~100 | 80% |
| 21~30 (金丹) | 105~180 | 85% |
| 31~40 (元婴) | 155~280 | 85% |
| 41~50 (化神) | 205~400 | 90% |
| 51~60 (炼虚) | 255~540 | 90% |
| 61~70 (合体) | 305~700 | 90% |
| 71~80 (大乘) | 355~880 | 95% |
| 81~90 (渡劫) | 405~1,080 | 95% |

**任务奖励:**
| 任务类型 | 灵石奖励 | 仙玉奖励 |
|----------|---------|---------|
| 主线任务 | 等级×50 + 200 | 0 |

**坊市交易:**

## 5. 修炼效率公式

### 5.1 核心公式

```
实际效率 = 基础速度 × 灵根倍率 × 功法倍率 × 丹药倍率 × 灵气潮汐倍率 × 场景倍率
```

**基础速度:** 100 修为/分钟 (全局固定)

### 5.2 灵根倍率

灵根决定修炼资质，是角色创建时随机生成的核心属性。

| 灵根品质 | 倍率 | 出现概率 | 说明 |

| 极品灵根 | 1.5 | 10% | 优秀 |
| 天灵根 | 2.0 | 5% | 变异, 极稀有 |
| 变异灵根(雷/冰/风) | 1.8 | 5% | 特殊效果 |

**灵根特殊效果:**
| 变异灵根 | 特殊效果 |

> **附录: 设计原则**
> 1. 所有数值以5分钟为单位设计战斗节奏，BOSS战控制在3-10分钟。
> 2. 付费与免费玩家的效率差距控制在3倍以内，避免过度P2W。
> 3. 数值随等级平滑增长，避免属性膨胀失控。
> 4. 所有百分比加成采用乘算，避免边际效应消失。
> 5. 预留20%数值空间供后续版本扩展（如新增境界、新装备品质）。
